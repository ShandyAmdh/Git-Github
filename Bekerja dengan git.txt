#Git command lokal
1. git init
2. git add <nama file> / bisa langsung sekaligus git add . => gunanya untuk memasukkan file ke state area sebelum di commit
3. git status, untuk melihat status file di area mana
4. git commit, commit setelah state area
5. git config, untuk melakukan configurasi
   * git config --global user.email "email kita"
   * git config --global user.name "nama kita"
6. git branch, membuat cabang baru
7. git help, bantuan cs git

Ada 3 area pada repo git:
1. Working tree, folder tempat mulai bekerja
2. Staging area
3. History

Perintah command :
1. pwd => untuk melihat kita ada difolder mana
2. ls => untuk melihat semua isi folder
3. cd <namafoldertujuan>  => naik 1 folder
4. cd .. => mundur 1 folder
5. git status => melihat status file di area mana 
6. git commit -m "pesan kita" / bisa juga git commit -am "pesan kita" (tapi cara kedua ini jika folder hanya modified)
7. git branch => melihat ada berapa branch direpo
8. git branch <nama branch> => membuat branch
9. git log --all --decorate --graph / buat saja alias supayah tidak perlu panjang mengetik code caranya:
   * alias graph = "git log --all --decorate --graph" jadi tinggal ketik graph 
10. git checkout <namabranch> => pindah ke branch lain
11. git merge <namabranch> => menggabungkan beberapa branch dengan catatan harus kembali dulu ke branch master
12. git branch -d <namabranch> =>menghapus branch yang sudah di merge
13. git branch --merged => untuk melihat branch mana yang sudah di gabung
14. git branch -D <namabranch> => untuk menghapus branch meskipun belum dimerge
15. git clone <link remote>
16. git remote => mengecek remote
17. git config --list => melihat semua configurasi git
18. git remote -v => melihat data lebih detail tidak hanya namanya
19. git push => menaruh file/repo dari lokal ke github
20. git pull => mengambil file/repo dari github ke lokal
